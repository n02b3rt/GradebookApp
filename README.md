# GradebookApp

Aplikacja typu Elektroniczny Dziennik Ocen stworzona w technologii WPF z wykorzystaniem Entity Framework Core oraz bazy danych PostgreSQL.

## Opis

GradebookApp umożliwia nauczycielom zarządzanie uczniami, wystawianie ocen oraz przeglądanie danych z relacyjnej bazy danych. Aplikacja pozwala na konfigurację połączenia z bazą danych przy pierwszym uruchomieniu i zapisuje ustawienia w pliku `appsettings.json`.

## Technologie

- .NET 8
- C#
- WPF (MVVM)
- Entity Framework Core
- PostgreSQL
- JSON (konfiguracja połączenia)

## Wymagania

- .NET SDK 8.0 lub nowszy
- PostgreSQL 14+
- Visual Studio 2022 (zalecane)
- Konto użytkownika PostgreSQL z uprawnieniami do tworzenia tabel
